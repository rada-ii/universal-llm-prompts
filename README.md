## Pro Tips

### **For Best Results:**

- **Be specific** → "Marketing plan for B2B SaaS targeting healthcare" vs. "marketing plan"
- **Include context** → "For busy executives who need quick insights"
- **Use examples** → "Similar to how Uber connects drivers and riders"
- **Set constraints** → "Budget under $50K, timeline 3 months"

### **For Structured Prompts:**

- **Set temperature to 0.1** for consistent JSON/CSV output
- **Copy entire prompt** - don't modify structured prompts
- **Validate results** with JSON/CSV parsers
- **Try Claude first** for complex data extraction

## Common Issues & Quick Fixes

| Problem                               | Quick Fix                                       |
| ------------------------------------- | ----------------------------------------------- |
